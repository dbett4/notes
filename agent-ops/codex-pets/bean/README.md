# Bean Codex Pet

**As of:** 2026-05-12

Install path:

```text
~/.codex/pets/bean/
```

Restore by copying `pet.json` and `spritesheet.webp` into that folder, then restart Codex if Bean is already loaded.

Files:
- `pet.json` - Codex custom pet manifest.
- `spritesheet.webp` - Final animated Bean atlas.
- `contact-sheet.png` - QA preview of all supported Codex pet states.
